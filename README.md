# 📍 Live Location Tracker

A real-time location tracking web app built using **Node.js**, **Express.js**, **Socket.io**, and **EJS**. Users can share their live location which updates automatically on a map using WebSockets and Leaflet.js.

---

## 🚀 Features

- 📡 Live location sharing
- 🗺️ Interactive map with Leaflet.js
- 🔄 Real-time updates using Socket.io
- 🌐 Uses browser's Geolocation API
- 🧩 EJS template engine for server-side rendering

---

## 🛠 Tech Stack

- **Backend:** Node.js, Express.js
- **Frontend:** EJS templating
- **Real-Time:** Socket.io
- **Maps:** Leaflet.js + OpenStreetMap
- **Geolocation:** HTML5 Geolocation API
